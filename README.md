# vento

这是一个hexo主题博客，主打一个简洁，后续可能会和服务端同构相结合，例如nuxt

## 安装

1. 下载源码包
```bash
cd blog # cd 到你的博客目录
git clone https://github.com/lovelyJason/hexo-them-vento
yarn
```

2. 将你blog/_config.yml中的theme改为vento


## highlightjs

按照官方文档介绍，默认配置下的代码高亮是不会工作的，要手动引入相应的主题css文件才可以
